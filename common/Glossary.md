## Glossary

- **Particle**: Rendered unit derived from an image pixel sample.
- **Home position**: Original target position for a particle based on the image layout.
- **Healing**: Spring‑like force returning particles to home.
- **Viscosity/Drag**: Damping applied to velocities to reduce oscillation.
- **Preset**: Saved set of parameters (forces, visuals) to reproduce an effect.
- **Impact preset**: Predefined force scenario (e.g., explosion, vortex).
- **Embed**: Self‑contained script/canvas snippet for third‑party sites.
- **Adaptive degradation**: Automatic reduction of particle count/effects to sustain FPS.
